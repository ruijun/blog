Nexus5刷官方CM13教程

发表于 2016-08-25 | 分类于 刷机 |

刷机前准备

  • 下载CM13 ROM
  • 下载Google Apps
  • 下载TWRP recovery

刷机步骤

刷TWRP recovery

1. 手机关机状态下，同时按住音量键下+电源键，进入recovery
2. 刷入TWRP recovery：fastboot flash recovery <recovery_image_file.name>
3. 重启系统：fastboot reboot

Clean/Wipe Data

1. 手机关机状态下，同时按住音量键下+电源键，进入recovery
2. 使用音量键上或下，选择recovery mode，按住电源键进入TWRP recovery
3. 依次选择 Wipe->Advanced Wipe

4. 步骤3完成后，进入selected partitions to wipe，选择Dalvik Cache, Data, Cache, System进行wipe

5. 完成后reboot system

备份数据

1. 进入TWRP recovery，选择Backup

2. 步骤1完成后，会进入下面的界面，选择你需要备份的数据进行备份即可

刷CM13 ROM

1. 复制GApps和CyanogenMod ROM压缩文件到你的设备上
2. 按照上面的步骤进入TWRP recovery
3. 依次install CyanogenMod ROM和GApps
4. 步骤3完成后Reboot system
5. Reboot system完成后，等待开机动画完成后，按照提示设置手机系统就好了。到这里就刷机完成了。

如何Root?

1. 进入开发者模式，点击Root授权

2. 步骤1完成后，会弹出Root授权对话框，选取应用和ADB即可

问题

卡在正在核对信息不动了

  • 原因：出现这个问题，是因为被墙了，导致连接不上服务
  • 解决方法：直接关机，再开机，然后到wifi那个界面选择跳过，不连接wifi
